# Test Prodigybuild

This repository is a comprehensive collection of classic data structures and algorithms implemented in C. It includes everything from basic algorithms like bubble sort to more complex structures like Binary Search Trees (BST) and Doubly Linked Lists.

## Setup On MAC

### Install GCC and Make

To work with this project, you must have `gcc` and `make` installed on your machine.

```bash
brew install gcc
```
```bash
brew install make
```

### Install Git

```bash
brew install git
```
Clone the repository

```bash
git clone https://github.com/user/test-prodigybuild.git
```
## Setup On Windows

### Installing GCC and Make

We recommend you install [Mingw-w64](http://mingw-w64.org/doku.php) via [MSYS2](https://www.msys2.org/).

### Installing Git
You can download Git from [Git Official Site](https://git-scm.com/)

Clone the repository

```cmd
git clone https://github.com/user/test-prodigybuild.git
```

## Running the Project

Navigate to the cloned repository and compile the C files with:

```bash
gcc filename.c -o outputfile
```

And run the program with:

```bash
./outputfile
```

## Environment Setup

The programming environment for this project is based on `gcc`. Make sure your setup is compatible with this.

## Deployment

The project does not have explicit deployment steps as it primarily consists of standalone C programs. However, any continuous integration or deployment steps will be executed as per the settings and configurations in the CI/CD pipeline.

Replace `filename` and `outputfile` with the name of the C file you wish to compile and the name of the output file respectively.
